Code Skills

Javascript

The internet is composed of over 90% Javascript, it is the key to building about anything useful on earth. Javascript integrates with everything, no matter if you are programming hardware or software, front end or back end.

Databases

Databases are used for storing and retrieving some sort of data after your application has been shut down. Mongod is the new go-to database for advanced scripting languages this is why we use it. Fast query access, easy to scale and less verbose code.

HTML5/CSS3

Combining HTML5 with CSS3 we can achieve advanced styling features that one could never achieve before. Web styling has become way more cross device compatable because of smartphones(smaller devices), further the markup is cached in the browser locally for quick loading later.

NodeJs

NodeJS is and open source server framework that uses Javascript for both front end development, its great for full stack web development for this reason. Further it is one of the fastest growing communities online meaning there is endless support for it.
